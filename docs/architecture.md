# 架构边界

## 添加功能

新功能放入 `modules/<name>`，拥有自己的 API、应用层、领域层、基础设施适配器和测试；然后在组合根注册 `ModuleDefinition`。跨模块通信应增加公开端口或领域事件。

## 删除功能

先通过 `FEATURE_<NAME>=false` 停用并验证，再删除组合根注册、模块目录和独立迁移。其他模块不得依赖它的内部实现或数据表。

## AI 决策链

1. 查询标准化缓存。
2. 在租户边界内检索证据。
3. 仅保留超过阈值的证据。
4. 无证据时拒答或转人工。
5. 模型只能基于证据生成答案。
6. 返回文档编号、来源和得分。

生产环境应增加混合检索、重排、答案数字校验、提示词注入检测和离线评测集。业务事实（订单、余额、物流）必须来自业务 API，不允许由模型推断。
