# 文档控制与交付说明

| 项目 | 内容 |
|---|---|
| 项目名称 | 星选商城企业智能客服系统 |
| 交付版本 | V0.1.0 |
| 文档基线 | 2026-06-30 |
| 技术基线 | FastAPI、SQLAlchemy、PostgreSQL/SQLite、Redis、OpenAI 兼容模型网关 |
| 交付状态 | 开发验证版，具备完整业务骨架，不等同生产终验 |

## 交付物

- `src/customer_service`：后端、AI 编排及内置管理页面。
- `tests`：接口、租户隔离、模型降级、权限和持久化测试。
- `docker-compose.yml`、`Dockerfile`：容器化运行配置。
- `.env.example`：环境变量模板，不含密钥。
- 本目录 11 份项目交付文档。

## 文档使用规则

所有配置、账号、密钥和生产地址以实施现场为准。文档变更应记录版本、修改人、日期和审批人；接口或数据库结构变更须同步修改 03、04、06 三份文档。严禁将生产 `.env`、API Key、数据库密码提交到版本库。

## 已知边界

默认 `AUTH_ENABLED=false`、`AI_MODEL_PROVIDER=mock`；电商订单为开发适配器；已建立 Alembic 初始迁移基线；Redis 已接入缓存和限流并支持故障降级；自动浏览器验收环境不可用，已以 HTTP 与自动化测试替代。
