# 测试计划与测试用例

## 测试范围

覆盖健康检查、模块装配、知识库 CRUD、混合检索、Rerank、证据门控、模型降级、订单事实、HTTP OMS、会话、工单、客户、持久化、认证、RBAC、租户隔离、审计、限流、安全拒答和 Prometheus 指标。

| 编号 | 场景 | 预期结果 |
|---|---|---|
| TC-01 | 无证据问答 | 拒答且不生成虚构引用 |
| TC-02 | 订单物流查询 | 返回 OMS 事实、订单号和引用 |
| TC-03 | 跨租户查询 | 返回 403 或 404 |
| TC-04 | 模型或 Rerank 服务异常 | 自动降级且不产生无依据回答 |
| TC-05 | 工单从待处理流转到解决 | 工单和会话状态同步 |
| TC-06 | 服务重启 | 知识、会话和审计记录仍然存在 |
| TC-07 | 错误密码或无效令牌 | 返回 401 |
| TC-08 | viewer 执行写操作 | 返回 403 |
| TC-09 | 管理员写操作 | 生成审计日志 |
| TC-10 | 获取其他用户订单或提示词注入 | 安全拒答且 grounded=false |
| TC-11 | Prometheus 指标检查 | `/metrics` 返回请求计数和延迟指标 |
| TC-12 | 180 条电商客服评测集 | 生成分类准确率、安全率和延迟报告 |
