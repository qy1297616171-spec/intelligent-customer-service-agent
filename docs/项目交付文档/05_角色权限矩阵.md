# 角色权限矩阵

| 能力 | owner | admin | editor | agent | viewer |
|---|:---:|:---:|:---:|:---:|:---:|
| 查看会话/客户/知识/分析 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 发起问答/处理会话 | ✓ | ✓ | ✓ | ✓ | — |
| 转人工及工单流转 | ✓ | ✓ | ✓ | ✓ | — |
| 编辑知识库 | ✓ | ✓ | ✓ | — | — |
| 编辑客户标签/状态 | ✓ | ✓ | ✓ | — | — |
| 成员及角色管理 | ✓ | ✓ | — | — | — |
| 查看审计日志 | ✓ | ✓ | — | — | — |
| 系统所有权配置 | ✓ | — | — | — | — |

权限执行采用“默认拒绝、最小授权”。JWT 携带用户、租户和角色，有效期默认 480 分钟；服务端同时校验请求 `tenant_id`。生产建议 owner 仅用于治理，日常使用 admin/agent；高权限账号启用 MFA，离职账号立即停用。
