# 项目总结与移交说明

## 已完成

已形成模块化电商客服骨架，完成客户、订单、会话、知识、混合检索、AI 编排、专业 Rerank、工单、分析、登录、RBAC、成员和审计能力；支持 PostgreSQL pgvector HNSW、SQLite 降级、Redis 缓存与限流、真实语义嵌入及模型失败降级。已接入独立 HTTP Mock OMS、Prometheus、Grafana、JSON 结构化日志、GitHub Actions 和 Docker Compose。当前自动化测试 29 项通过。

## 上线前必须完成

1. 在 `.env.production` 注入真实 LLM、Embedding、Rerank 密钥并执行 180 条评测集，不在文档或仓库中传递密钥。
2. 将 `OMS_BASE_URL` 和凭据切换到真实 OMS/ERP/WMS，并完成联调验收；当前已提供标准 HTTP 适配器和独立 Mock OMS。
3. 获得目标服务器、域名和部署权限后，通过 `Deploy Demo` 工作流发布公网 Demo，并配置 HTTPS。
4. 在生产候选环境完成渗透、灾备恢复、灰度、72 小时稳定性和真实模型端到端压测。
5. 轮换全部演示密码/密钥，配置告警通知渠道、日志集中化、Redis 高可用及值班机制。
